# brand-spec.md — Claude Code 安装引导门面

## 品牌锚点
本页是 Claude Code（Anthropic）的引导物料，视觉直接从 Claude/Anthropic 品牌语言生长，而非通用 SaaS 模板。

## 色板（从 Anthropic/Claude 视觉语言提取）
- 纸感底色 paper：`#F7F4EE` / 更浅 `#FBFAF7`
- 主墨 ink：`#1A1915`
- 次级文字 muted：`#6B6557`
- 品牌粘土橙 clay（accent）：`#CC785C`（深 `#B5654A`）
- 暖描边 line：`#E7E1D5`
- 代码块暖炭底：`#2B2622`，文字 `#EFE9DD`（**不用** GitHub 深蓝 + 霓虹的烂大街解）
- 成功绿 ok：`#3F7D54`

## 字型
- Display（英文/标题）：Newsreader（serif，Google Fonts，离线优雅回退 Georgia）
- CJK 标题与正文：系统 PingFang SC / Microsoft YaHei（中文无合适免费 serif，系统字最干净）
- 代码：ui-monospace / Menlo / Consolas

## 品牌母题
- Anthropic 的星芒/星号意象 → 用纯 CSS/SVG 画一个克制的「✳ 星芒」做装饰母题，不伪造官方 logo。

## 气质关键词
温暖、可信、安静、亲切、不吓人。给「怕电脑」的人一种「有人陪着、一步步来」的安心感。

## 反 slop 自检
- ❌ 紫渐变、❌ GitHub 深蓝+霓虹、❌ emoji 当唯一图标语言、❌ 圆角卡+左彩条
- ✅ 暖纸底、单一 clay accent 贯穿、serif display、`text-wrap: pretty`、oklch/既有色不乱发明
