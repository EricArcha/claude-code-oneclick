Claude Code 安装引导
给「电脑小白 / AI 新手」的保姆级引导

把 Claude Code 装进你自己的电脑

不用懂编程。跟着下面三步，复制 → 粘贴 → 回车，你就能拥有一个住在电脑里、能帮你动手干活的 AI 助手。

开源仓库：github.com/EricArcha/claude-code-oneclick
为什么是 Claude Code

不是又一个聊天框，是会动手的助手

它能直接动手

网页版 AI 只能聊天，你得来回复制粘贴。Claude Code 直接读写你电脑里的文件、帮你跑命令。

它会自己干多步

不是一问一答，而是能连续完成一整件任务的「智能体」：自己查、自己改、自己验证。

它可换、可扩展

后台 API 随你换，还能装联网搜索、飞书等插件。比封闭的网页产品自由得多。

「网页 AI 像隔着玻璃指挥；Claude Code 是请进门，帮你动手。」

安装最新版 Claude Code

一行命令搞定。原生安装版以后会自动更新，无需手动升级。

2

配置你自己的 API

填上你的服务商地址和密钥，就不需要官方账号登录了。随时可重跑换服务。

脚本会把配置安全地写进 ~/.claude/settings.json，不会动你其他设置；密钥不会显示在屏幕上。想换一家服务商？再运行一次这个命令就行。
3

给它装上「联网搜索」

让 Claude Code 能查最新资料。两家免费服务，配一个或都配都行。

T

Tavily

实时网页检索，适合「查最新 / 找事实」。免费每月 1000 次、免信用卡。

注册：app.tavily.com
E

Exa

语义搜索，按「意思」找相似内容与研究资料，适合「做研究」。新号送 $10 额度。

注册：dashboard.exa.ai
官网打不开拿不到 Key？别卡在这——先跳过，本地备份说明在 docs/tavily.md、docs/exa.md，回头再配。
