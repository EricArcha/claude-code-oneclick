Before you start · 30 秒预备课

先学会「打开终端」

下面每一步，都是把一行命令复制进一个叫「终端」的小窗口、再按回车。终端就是让你用文字给电脑下指令的地方——它不会咬人，照着做就行。先按你的系统把它打开：

macOS —— 叫「终端 / Terminal」

按住 ⌘ Command 再按 空格，弹出搜索框，输入「终端」或「terminal」，回车。一个小窗口就打开了。

Windows —— 用「PowerShell」

按键盘左下角 ⊞ Win 键（窗户图标），直接输入「powershell」，点出现的「Windows PowerShell」打开即可。

▶
怎么运行命令？点命令框右上角「复制」→ 切到终端窗口粘贴（按 ⌘ V）→ 按 回车。就这么简单。
▶
怎么运行命令？点命令框右上角「复制」→ 切到 PowerShell 窗口粘贴（点右键，或按 Ctrl V）→ 按 回车。就这么简单。
?
听过 cmd（命令提示符）吗？它也是个命令窗口，但本引导的命令请统一用 PowerShell，更顺、更少坑。
